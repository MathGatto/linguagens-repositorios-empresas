# PROJETO:

  Projeto de construção de um pipeline de dados ETL através da biblioteca REQUESTS.

  - Tratamos e transformamos os mesmo em Dataframes com PANDAS
  - Salvamos em formato CSV e disponibilizamos ao time de BI
